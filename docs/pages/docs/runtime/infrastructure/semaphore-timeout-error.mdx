Error thrown when a [Semaphore](/docs/runtime/infrastructure/semaphore#semaphore) acquire operation times out while waiting in the queue.

Contains a `code` field (default: `'SEMAPHORE_QUEUE_TIMEOUT'`) for programmatic error handling.
In the context of [AgentRuntime](/docs/runtime/agent-runtime/#agentruntime), this maps to the `TOOL_QUEUE_TIMEOUT` error code when
a tool cannot acquire a concurrency slot within `toolExecutionQueueTimeoutMs`.

class SemaphoreTimeoutError extends Error {
