Defines a context entry that is sent with every agent request.

Context entries provide the agent with information about the current application state
(e.g., current page, user preferences, selected items). The `persistent` flag controls
whether the context survives component unmounting (default: true).

Related: [useAgentContext](/docs/react-core/hooks/use-agent-context#useagentcontext) registers context, [frontendContextAtom](/docs/react-core/state-management/frontend-context-atom#frontendcontextatom) stores it,
[ToolContext](/docs/common/tools/tool-context#toolcontext).frontendContext receives it on the server.

```ts
interface ContextDefinition {
    key: string;
    description: string;
    value: unknown;
    persistent?: boolean;
    priority?: number;
}
```
