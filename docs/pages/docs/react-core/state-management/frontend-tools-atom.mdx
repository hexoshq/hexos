Registry of frontend tools and actions available for local execution.

Populated by [useAgentTool](/docs/react-core/hooks/use-agent-tool#useagenttool) and [useAgentAction](/docs/react-core/hooks/use-agent-action#useagentaction) hooks.
When the LLM calls a tool whose name matches an entry in this map,
[useAgent](/docs/react-core/hooks/use-agent#useagent) executes it locally instead of waiting for a server result.
