Indicates whether the agent is currently streaming a response.

Set to `true` on `message-start` events and `false` on `stream-complete` or `error`.
Used by [StreamingIndicator](/docs/react-ui/ui-components/streaming-indicator#streamingindicator) and [InputComposer](/docs/react-ui/ui-components/input-composer#inputcomposer) to show loading state
and disable input during streaming.
