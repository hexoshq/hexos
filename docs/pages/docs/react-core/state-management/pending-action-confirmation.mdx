Represents a pending action confirmation waiting for user decision.

Created when an action with `confirmationMessage` is triggered. The `resolve`
function is called with the user's decision (true to confirm, false to cancel).

Related: [ActionConfirmDialog](/docs/react-ui/ui-components/action-confirm-dialog#actionconfirmdialog) renders the confirmation UI,
[pendingActionConfirmationsAtom](/docs/react-core/state-management/pending-action-confirmations-atom#pendingactionconfirmationsatom) stores these entries.

```ts
interface PendingActionConfirmation {
    id: string;
    actionName: string;
    args: unknown;
    message: string;
    resolve: (confirmed: boolean) => void;
}
```
