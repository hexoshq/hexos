Represents a pending tool approval request on the frontend.

Created when an `approval-required` [TransportEvent](/docs/react-core/transport/transport-event#transportevent) arrives from the server.
Contains the tool call metadata needed to display an approval dialog to the user.

Related: [useToolApproval](/docs/react-core/hooks/use-tool-approval#usetoolapproval) manages these requests, [ToolApprovalDialog](/docs/react-ui/ui-components/tool-approval-dialog#toolapprovaldialog)
renders the approval UI, [ApprovalDecision](/docs/react-core/tools/approval-decision#approvaldecision) is the user's response.

```ts
interface ToolApprovalRequest {
    toolCallId: string;
    toolName: string;
    args: unknown;
    agentId: string;
    timestamp: Date;
}
```
