Frontend representation of a message in the conversation, with structured parts for rich rendering.

Includes a `parts` array for granular rendering of tool calls, reasoning blocks, and images.
The `metadata` field allows attaching arbitrary data for custom rendering.

Related: [MessagePart](/docs/react-core/core-types/message-part#messagepart) defines part types, [messagesAtom](/docs/react-core/state-management/messages-atom#messagesatom) stores the array,
[MessageBubble](/docs/react-ui/ui-components/message-bubble#messagebubble) renders individual messages.

```ts
interface AgentMessage {
    id: string;
    role: 'user' | 'assistant' | 'system' | 'tool';
    content: string;
    createdAt: Date;
    agentId?: string;
    metadata?: Record<string, unknown>;
    parts?: MessagePart[];
}
```
