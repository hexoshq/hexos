Server-Sent Events transport implementation for agent communication.

Default transport that works through all CDNs and proxies. Implements the
[AgentTransport](/docs/react-core/transport/agent-transport#agenttransport) interface using HTTP POST requests with streaming
response bodies parsed as SSE (`data: \{json\}\n\n` lines).

Supports dynamic headers via async functions (e.g., refreshing JWT tokens),
request cancellation via AbortController, and configurable approval endpoints
for human-in-the-loop workflows.

Related: [AgentConfig](/docs/react-core/transport/agent-config#agentconfig) configures the connection, [useAgent](/docs/react-core/hooks/use-agent#useagent) creates
and manages the transport instance, [TransportEvent](/docs/react-core/transport/transport-event#transportevent) defines the event types
consumed from the SSE stream.

```ts
class SSETransport implements AgentTransport {
    connect(config: AgentConfig) => Promise<void>;

    sendApproval(decision: ApprovalDecision) => Promise<void>;
    onMessage(callback: (event: TransportEvent) => void) => () => void;
    disconnect() => void;
}
```
