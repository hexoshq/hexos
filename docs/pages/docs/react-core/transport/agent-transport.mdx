Abstract transport interface for client-server communication.

[SSETransport](/docs/react-core/transport/ssetransport#ssetransport) is the default implementation using HTTP Server-Sent Events.

Related: [AgentConfig](/docs/react-core/transport/agent-config#agentconfig) configures the connection, [TransportMessage](/docs/react-core/transport/transport-message#transportmessage) is
the outbound payload, [TransportEvent](/docs/react-core/transport/transport-event#transportevent) is the inbound event type.

```ts
interface AgentTransport {
    connect(config: AgentConfig): Promise<void>;
    send(message: TransportMessage): void;
    sendApproval(decision: ApprovalDecision): Promise<void>;
    onMessage(callback: (event: TransportEvent) => void): () => void;
    disconnect(): void;
}
```
