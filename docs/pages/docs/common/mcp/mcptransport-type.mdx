Discriminator for MCP server transport protocols.

MCP servers can communicate via `stdio` (local process) or `sse` (HTTP Server-Sent Events).
Used by [MCPServerConfig](/docs/common/mcp/mcpserver-config#mcpserverconfig) to determine the transport type.

```ts
type MCPTransportType = 'stdio' | 'sse'
```
