Contains the tool name, optional description, and its JSON Schema input definition.
Retrieved by [MCPManager](/docs/runtime/mcp/mcpmanager#mcpmanager) during server initialization and cached for reuse.

Related: [MCPServerConfig](/docs/common/mcp/mcpserver-config#mcpserverconfig) configures the server, [ToolDefinition](/docs/common/tools/tool-definition#tooldefinition) is the
runtime-native equivalent.
