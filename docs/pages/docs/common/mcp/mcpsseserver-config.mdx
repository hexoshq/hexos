Configuration for an MCP server communicating via HTTP Server-Sent Events.

Connects to a remote MCP server over HTTP. Supports custom headers for authentication
(e.g., Bearer tokens). Suitable for shared or cloud-hosted MCP servers.

Related: [MCPServerConfigBase](/docs/common/mcp/mcpserver-config-base#mcpserverconfigbase) provides shared settings,
[MCPStdioServerConfig](/docs/common/mcp/mcpstdio-server-config#mcpstdioserverconfig) is the alternative for local servers.

```ts
interface MCPSSEServerConfig extends MCPServerConfigBase {
    transport: 'sse';
    url: string;
    headers?: Record<string, string>;
    timeout?: number;
}
```
