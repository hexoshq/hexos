Abstract transport interface for client-server communication in Hexos.

Defines the contract for connecting to a backend, sending messages, receiving events,
and disconnecting. [SSETransport](/docs/react-core/transport/ssetransport#ssetransport) is the default implementation using HTTP
Server-Sent Events. Future implementations could support WebSockets or other protocols.

Related: [SSETransport](/docs/react-core/transport/ssetransport#ssetransport) implements this interface, [AgentConfig](/docs/react-core/transport/agent-config#agentconfig) configures
the connection, [TransportMessage](/docs/react-core/transport/transport-message#transportmessage) is the outbound payload,
[TransportEvent](/docs/react-core/transport/transport-event#transportevent) is the inbound event type.

```ts
interface AgentTransport {
    connect(config: AgentConfig): Promise<void>;
    send(message: TransportMessage): void;
    onMessage(callback: (event: TransportEvent) => void): () => void;
    disconnect(): void;
}
```
