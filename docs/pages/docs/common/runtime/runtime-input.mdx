Input payload for a conversation turn, containing the user message and context.

Passed to [AgentRuntime](/docs/runtime/agent-runtime/#agentruntime).stream() or invoke() to initiate a conversation turn.
The `context` field carries arbitrary frontend data that becomes available to tools
via [ToolContext](/docs/common/tools/tool-context#toolcontext).frontendContext.

Related: [RuntimeOutput](/docs/common/runtime/runtime-output#runtimeoutput) is the non-streaming response,
[RuntimeEvent](/docs/common/core-types/runtime-event#runtimeevent) is the streaming response.

```ts
interface RuntimeInput {
    message: string;
    conversationId: string;
    context?: Record<string, unknown>;
    userId?: string;
}
```
