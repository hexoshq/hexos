Available Ollama model identifiers for use with `LLMProvider.Ollama`.

These values map to locally-hosted models via the Ollama runtime.
Custom models can also be specified as a plain string in [ModelConfig](/docs/common/agent-config/model-config#modelconfig).

```ts
enum OllamaModel {
    Llama3 = 'llama3'
    Llama31 = 'llama3.1'
    Llama32 = 'llama3.2'
    Mistral = 'mistral'
    Mixtral = 'mixtral'
    CodeLlama = 'codellama'
    Phi = 'phi'
    Gemma = 'gemma'
    Qwen = 'qwen'
}
```
