File or media attachment associated with a message.

Supports image, file, and audio types. The `data` field contains the content
as a base64 string or data URI. Attachments are sent with user messages via
[TransportMessage](/docs/react-core/transport/transport-message#transportmessage) and stored in [AgentMessage](/docs/react-core/core-types/agent-message#agentmessage).

```ts
interface Attachment {
    type: 'image' | 'file' | 'audio';
    name: string;
    data: string;
    mimeType: string;
}
```
